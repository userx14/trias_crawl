Datenlizenz und Datennutzung

Auf dieser Seite wird ein verarbeiter Datensatz erstellt aus Echtzeit-Fahrplandaten des ÖPNV Baden-Württemberg (MOBIDATA BW) bereitgestellt. Die hier bereitgestellten Daten fallen unter die Lizenz DL-DE->BY-2.0.

Quellcode und Datenverfügbarkeit

Der gesamte Code zur Abruf und Analyse der Daten ist auf github einsehbar. Für die Echtzeit-Datenabfrage bei Mobidata-BW ist jedoch ein API-Key erforderlich, der nicht weitergegeben werden darf. Aktuelle Live-Daten stehen als json-Datensatz direkt zum Abruf bereit. Historische Daten können aufgrund ihrer Größe nicht direkt angeboten werden. Bei Interesse bitte per Mail anfragen, der aktuelle Datensatz reicht bis Anfang 2026 zurück.
